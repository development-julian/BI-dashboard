# **App Name**: InsightBoard MVP

## Core Features:

- Basic Layout: Provides a basic layout structure with sidebar, header, and content area for the main dashboard and detail view.
- Reusable Metric Cards: Displays key performance indicators (KPIs) using mock data in a consistent and visually appealing format.
- Basic Charts: Offers 2-3 chart types (e.g., line, bar) with mock data for data visualization in the main dashboard and detail view.
- Simple Sortable Table: Presents mock data in a table format with basic sorting capabilities for the detail view.
- Main Dashboard (Overview): Displays a high-level overview of key metrics and charts using the reusable metric cards and basic charts.
- Simple Detail View: Provides a drill-down view from a metric or chart, displaying a simple sortable table with supporting data and the original chart/metric.
- Minimal Settings Screen: Visual placeholder for a settings screen, allowing future implementation of user preferences and customization. Does not implement any settings logic.
- Themed Dashboard Selection Placeholder: A placeholder that represents the future incorporation of a dashboard suggestion tool. LLM tool suggests dashboards and relevant metrics given some user input.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5), provides a sense of stability and insight.
- Background color: Very light gray (#F5F5F5), offers a clean and neutral backdrop.
- Accent color: Cyan (#00BCD4), brings a vibrant and modern feel for interactive elements.
- Headline font: 'Space Grotesk', sans-serif, for headlines and titles.
- Body font: 'Inter', sans-serif, for body text and labels. This font pairing offers a clean and modern aesthetic, and provides high legibility.
- Use a consistent set of minimalistic icons from a library like Material Icons to represent different metrics and functions.
- Implement a responsive grid system to ensure the dashboard adapts to different screen sizes.
- Incorporate subtle transitions and animations for user interactions, such as loading states.
